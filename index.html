Рукоятка релейная РР

Релейная рукоятка с дискретными контактами

Рукоятка релейная управления с дискретными контактами (сухой контакт). Не имеет пропорционального сигнала, работает по принципу контакт замкнут/разомкнут при отклонении рукоятки. Идеальна для систем с релейным управлением.

  • Дискретное управление «сухой контакт»
  • Релейная логика без пропорционального сигнала
  • Высокая надежность релейной коммутации
  • Простота интеграции в релейные системы
  • Настройка под требования заказчика

КПС

Контроллер преобразования сигналов

Универсальный контроллер для электрогидравлических систем

Контроллер предназначен для преобразования аналоговых сигналов или цифрового сигнала из шины CAN в сигналы ШИМ для управления различными устройствами (золотники гидрораспределителей, гидронасосы, гидромоторы и пр.). Все выходы оснащены измерителями тока с ПИД-регулятором для точного поддержания заданных значений. Специализированное ПО позволяет гибко настраивать контроллер.

  • Преобразование CAN в ШИМ
  • Встроенный ПИД-регулятор
  • Измерение тока на всех выходах
  • Входы Lock_A/Lock_B для безопасности
  • Гибкая настройка параметров
